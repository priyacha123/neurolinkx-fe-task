# Neurolinkx | Shipment Tracking Dashboard

A premium, production-quality SaaS dashboard built with the latest React ecosystem.

![alt text](image.png)

## 🚀 Features

### 📦 Reusable UI Library
- **Accessible**: Built on Radix UI primitives.
- **Themed**: Full dark mode support using Tailwind CSS 4.
- **Polished**: Consistent design tokens, shadows, and micro-animations.
- **Components**: Button, Table, Dialog, Toast, Command Palette (Cmd+K), and more.

### 🚢 Shipment Management
- **Dashboard**: Real-time KPI tracking and interactive charts.
- **Tracking**: Dynamic shipment detail pages with animated timelines.
- **Management**: Searchable, sortable shipment tables with bulk actions.
- **Settings**: Profile management, team invitations, and preferences.

### 🛠 Engineering Quality
- **Type Safe**: 100% TypeScript.
- **Tested**: Unit tests with Vitest & E2E tests with Playwright.
- **CI/CD**: Automatic linting, testing, and building via GitHub Actions.
- **Performant**: Next.js 15 App Router with optimized asset loading.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS 4
- **State**: Zustand
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Testing**: Vitest, Playwright

## 🏁 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Run Tests**:
   ```bash
   npm test
   ```

4. **Run E2E Tests**:
   ```bash
   npx playwright test
   ```
